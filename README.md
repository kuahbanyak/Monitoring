# Monitoring

## How To RUn

1. Clone this repository
   ```
   $ git clone https://github.com/kuahbanyak/Monitoring.git
   ```
2. Install all depedencies on the package.json
   ```
      cd Monitoring/fe
      sebelum itu cek package.json dlu 
      npm install
      cd Monitoring/be
      node index
    
   ```
3. Run web
   ```
   $ npm start
   ```
